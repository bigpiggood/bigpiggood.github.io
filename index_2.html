studio_reservation
(web_app)




Development Detail :

개발 기획


- OTT플랫폼 이용자의 증가에 따라 영상콘텐츠 소비가 지속적으로
  늘고 있음.

- 이에 콘텐츠 생산을 위한 소-대형 전문 스튜디오 예약도
  늘고 있으나 현재 예약/결제 관련 중개 플랫폼은 거의 전무함.

- 따라서 해당 웹 앱을 통해 스튜디오와 이용자간의 B2C망을 형성,
  업무 효율을 도모하고 편리함을 제공하고자 함.

주요 기능


- 웹 앱 기본 기능 (회원 가입 / 로그인 / 비밀번호 변경 등)


- 가격 / 지역 / 별점 순 등 정렬 기능


- 스튜디오 등록 및 관리자 승인 / 취소 기능


- 예약 및 결제 기능


- 이용자 리뷰(별점 후기 등) 작성 기능

사용 기술


- html/css/javascript(jQuery 라이브러리 및 위젯)


- Python : flask(프레임워크), pandas(라이브러리), jinja2(템플릿엔진)

- API : 구글(smtp), 네이버/카카오(소셜 로그인), 다음(주소 검색)
          결제시스템(포트원-PG통합연동-카카오페이)

- Mysql : 서버 데이터 관리
